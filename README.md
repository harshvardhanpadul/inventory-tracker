# Inventory Tracker
INTERN ID: CITS1980
An inventory management application for storing and tracking products.

## Features
- Add products
- Edit products
- Delete products
- Search products
- Quantity tracking
- Low stock indication
- Bootstrap interface

## Tech Stack
- Node.js
- Express.js
- PostgreSQL
- EJS
- Bootstrap

## Run Locally

```bash
npm install
nodemon app.js
```
